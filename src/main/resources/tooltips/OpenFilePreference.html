Open the test file
This option specifies where Squaretest should open the test file.

In the active editor window
Squaretest will always open the test file in the active editor window (the editor window
containing the source class).

In the next editor window if one is available
Squaretest will open the test file in the editor window next to the active editor window if one is available;
otherwise, it will open the test file in the active editor window.

In the next editor window, creating one if needed
Squaretest will open the test file in the editor window next to the active editor window if one is available;
otherwise, it will create a new editor window by splitting the active editor window vertically and open the
test file in the new window.

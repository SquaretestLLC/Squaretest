Data Constructor Limit When Bean Option Available
Specifies the longest, nested constructor call (number of parameters) that will be used to initialize recognized data classes.
If the constructor call exceeds this limit Squaretest will use the zero-argument constructor and bean
setter methods to initialize the class, if one is available.

Recognized Data Classes
Squaretest considers a class to be a data class if it has at least one data annotation.
Examples include annotations with canonical names that start with "lombok",
"javax.xml.bind.annotation", etc.

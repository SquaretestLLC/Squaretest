Module Configuration
This is the action taken the first time Squaretest generates a test in a module.

Automatically configure when possible
Squaretest will automatically configure settings for the module without prompt if the Project Settings
contain a template and the following are true.
    1. The module contains exactly one Test Sources Root, and the Test Sources Root contains source files matching the
         template language (Java or Groovy) in the Project Settings.
    2. The module test classpath contains the test frameworks and assertion libraries used in the project template.
    3. The module test classpath does not contain a more recent version of JUnit than the project template.
    4. The module test classpath does not contain Spring, AndroidJUnit4, Robolectric or AssertJ while the
         project template does not.

Always ask to configure module settings
Squaretest will always prompt the user to configure settings the first time a test is generated for a module.
